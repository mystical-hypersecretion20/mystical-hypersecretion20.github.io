🧾 recibbi-engine - Turn grocery photos into data insights

Download from GitHub

Recibbi Labs Receipt Insights turns a photo of your paper grocery receipt into a clean digital breakdown. You provide a photo from any major store, and the engine turns the messy ink into structured data. It identifies products, grabs prices, finds images, and generates a web link you can share with your family or friends.

📦 How It Works

The system uses professional visual scanning technology to read your receipts. It extracts line items and prices. Then, it uses web index tools to add images and extra data to your list. You get a clear summary through a Telegram bot or a direct command line tool.

⚙️ System Requirements

To run this tool on your Windows computer, you need a few things set up first:

  • Windows 10 or 11.
  • Docker Desktop installed on your system.
  • A stable internet connection for processing images.
  • A Telegram account if you choose to use the bot method.

💾 Setup and Installation

Follow these steps to get the software running on your computer:

  1. Visit the official download page.
  2. Download the latest release package.
  3. Save the folder to your computer and extract the contents.
  4. Open Docker Desktop and ensure it shows as running.
  5. Open your Windows command prompt inside the extracted folder.
  6. Type the command "docker-compose up" and press Enter.
  7. Wait for the text to stop scrolling; the service is now active locally.

📱 Using the Receipt Scanner

Once you start the engine, you have two ways to interact with it. The Telegram bot interface is the easiest choice for most users. You type the name of the store and upload your image. The engine will process the receipt and return a link to your digital item list within seconds.

🔒 Data Privacy

Your receipt data stays private. The engine processes your images to extract text and then creates a secure record. You control the shareable link. If you need to remove a receipt from the system, you can use the command line tools to delete the specific file or clear your local history.

🛠 Troubleshooting

If you encounter issues, check these steps:

  • Ensure Docker has enough memory allocated in the settings menu.
  • Confirm your internet connection allows access to standard web services.
  • Restart your computer if the containers fail to launch on the first attempt.
  • Re-download the folder if you receive errors about missing configuration files.
